调研icp&lp_icp点云配准方法的效果

我们在武汉大学真实激光扫描数据集（WHU-TLS，主要为山地场景）上，在完全相同的条件下同步运行 ICP 和 LP-ICP，从多个维度进行量化比较。我们在武汉大学真实激光扫描数据集（WHU-TLS，主要为山地场景）上，在完全相同的条件下同步运行 ICP 和 LP-ICP，从多个维度进行量化比较。

**实验 A（精度基准测试）**显示：传统 ICP 的各项误差远大于 LP-ICP，在初始位姿偏差大、重叠率低的山地场景下几乎无法完成配准任务；而 LP-ICP 凭借 FPFH 特征匹配，召回率高达 96.6%，代价是运行时间约为 ICP 的 2~3 倍。

**实验 B（旋转扰动鲁棒性）**表明：面对额外旋转扰动，ICP 完全失效，而 LP-ICP 保持了稳定较高的配准成功率，体现出其对初始位姿误差的强鲁棒性。

**实验 C（低重叠率鲁棒性）**揭示了 LP-ICP 的一个局限：当重叠率降至 0.8 以下时，粗配准内点数从 300 急剧下降至 0，粗配准失败，导致 LP-ICP 整体性能骤降，甚至不如传统 ICP。这说明 LP-ICP 的性能瓶颈在于其粗配准阶段对重叠率的依赖。

**实验 D（噪声鲁棒性）**显示：LP-ICP 在高噪声下仍能维持较高配准精度，但运行时间相比 ICP 大幅增加。原因在于：噪声增大时，RANSAC 需要更多迭代轮次才能找到足够的内点，揭示了 FPFH 对噪声的固有敏感性。
